## Actix Rust MongoDB Crud Example

![simplecrud](image.png)




## Usage

You will need rust and mongodb installed locally on your pc. 

- Clone the repo. 
- Cargo run. 
- Use Postman or Insomnia to test routes and get, put, delete, create books.
